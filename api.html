API

Parcel Insertion

API Document

Using this RESTful API every e-commerce can order parcel dynamically. You can directly call the url with your user id, api key and api secret. Ajax and CURL also can call the RESTful API

Required Parameters

Parcel
Item - parcel
Description - Define the type operation to be performed
Value - Insert
Type - String
Recipient Name
Item - recipient_name
Description - Parcel receiver's name not more than 255 characters
Value - Abdullah Al Mamun
Type - String
Recipient Mobile
Item - recipient_mobile
Description - Parcel receiver's mobile number exactly 11-digit long
Value - Recipient Mobile
Type - String
Recipient City
Item - recipient_city
Description - Parcel receiver's city not more than 40 characters
Value - Dhaka / Provide City Name from eCourier Coverage Cities
Type - String
Recipient Area
Item - recipient_area
Description - Parcel receiver's area not more than 40 characters
Value - Badda / Provide Area Name from eCourier Coverage Area
Type - String
Recipient Address
Item - recipient_address
Description - Parcel receiver's detail address not more than 120 characters
Value - House #34, Road #4, Block #4 Or Gudaraghat, North Badda Or Village: Rosulpur, Post: Post, Thana: Thana, Dist: District
Type - String
Package Code
Item - package_code
Description - Your Package Code
Value - #2423
Type - String
Product Price
Item - product_price
Description - Parcel price
Value - 1200
Type - Integer
Payment Method
Item - payment_method
Description - Payment Method
Value - Cash On Delivery - COD, Point of Sale - POS, Mobile Payment - MPAY, Card Payment - CCRD
Type - String

Optional Parameters

Recipient Landmark
Item - recipient_landmark
Description - Receiver's Landmark to find address efficiently - not more than 30 characters
Value - Near Fuzi Tower Default - N/A
Type - String
Parcel Type
Item - parcel_type
Description - Type of parcel
Value - For box - BOX (default), For documents - DOC, For gift - GIFT
Type - String
Is Anonymous
Item - is_anonymous
Description - If you don't want to expose yourself (for GIFT type)
Value - Anonymous - 1, Expose Identity - 0, Default - null
Type - Integer
Requested Delivery Time
Item - requested_delivery_time
Description - Requested delivery time
Value - 2017-02-01 (YYYY-MM-DD) Default Next Day
Type - String
Delivery Hour
Item - delivery_hour
Description - For fixed time base package provide a delivery time slot
Value - Anytime - any (default) 10:30am - 1:00pm, 1:00pm - 4:00pm, 4:00pm - 7:00pm
Type - String
Recipient Zip
Item - recipient_zip
Description - Receiver�s ZIP Code
Value - 1212, Default - null
Type - String
Product Id
Item - product_id
Description - Your Product or order id - not more than 20 characters
Value - GLX123 (must be unique if you want to track using this id instead of eCourier ID)
Type - String
Pick Address
Item - pick_address
Description - Your alternate pick address not more than 255 characters
Value - Any valid address, Default - null
Type - Integer
Comments
Item - comments
Description - Instructions for delivery not more than 255 characters
Value - Default - null
Type - String
